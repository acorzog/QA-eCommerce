Cypress tests implemented for an already existing app

4. **Install project dependencies using Yarn**

   ```bash
   yarn
   ```

5. **Start the web application**

   ```bash
   yarn start
   ```

6. **Run the tests**
  
   To open Cypress in interactive mode:
   
   ```bash
   yarn cypress:open
    ```

   To open Cypress in headless mode:

    ```bash
   yarn cypress run
    ```
    
   OR
   
    ```bash
   yarn test
    ```
